# Collaborative Note Taking Web App (MERN)

This is a collaborative note-taking web application built using the MERN stack (MongoDB, Express.js, React, Node.js).

## Features
- JWT authentication
- Create, edit, and delete notes
- Full text search
- Collaborator management
- Responsive UI using Tailwind CSS

## Tech Stack
Frontend: React, Tailwind CSS  
Backend: Node.js, Express.js  
Database: MongoDB
